In this folder, we have all the scripts which supports the running project.
1/2/4/5on(off) scripts are used to toggle the corresponding ports on beaglebone to turn on/off.
bone.cs is the C# script which is used to controll the beaglebone.
bone.exe is the executable file for bone.cs
pyScript1/2/4/5 are python scripts which are used for running UART communication between the beaglebone and Teensys.
